
# QUILL. 

Quill is a full-stack Notes application built with the MERN stack (MongoDB, Express.js, React, Node.js).
It allows users to dynamically create, update, and delete notes, providing a smooth and intuitive user experience.


## Features

- Add, read, update, and delete notes in real-time

- Clean and responsive React frontend

- RESTful API built with Node.js and Express.js

- Data persistence using MongoDB Atlas

- Proper use of HTTP methods (GET, POST, PUT, DELETE) for API operations


## Tech Stack

**Frontend:** React.js, Tailwind CSS, Daisy UI.

**Backend:** Node.js, Express.js

**Database:** MongoDB Atlas

**APIs:** RESTful APIs using standard HTTP methods


## Run Locally

Clone the project

```bash
  git clone https://github.com/preposterouspersona/Quill.git
```

Go to the project directory

```bash
  cd Quill
```

Install dependencies

```bash
  cd backend
  npm install
  cd ..
```
``` bash
  cd frontend
  npm install
  cd ..
```

Start the server

```bash
  cd backend
  npm run dev
```

Start the client

```bash
  cd frontend
  npm run dev
```
